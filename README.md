# Awesome AI CLI [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of AI-powered command-line interface tools that enhance developer productivity

AI CLI tools bring the power of large language models directly to your terminal, enabling code generation, explanation, debugging, and more without leaving your development environment.

## AI Assistant CLIs

### Amp CLI
**[Amp](https://www.amp.dev/)** - AI-powered development assistant for the command line.

**Licensing:** Proprietary

**Benefits:**
- Seamless integration with development workflows
- Context-aware code suggestions and completions
- Multi-language support for various programming languages
- Terminal-native experience with minimal context switching

---

### Claude Code
**[Claude Code (via Claude API)](https://www.anthropic.com/claude)** - Anthropic's Claude AI accessible through CLI interfaces.

**Licensing:** Proprietary (API-based)

**Benefits:**
- Long context windows for understanding large codebases
- Strong reasoning capabilities for complex coding tasks
- Safety-focused AI responses
- Multiple Claude models (Claude 3 Opus, Sonnet, Haiku) for different use cases
- Available through various community CLI wrappers

---

### Codex CLI
**[Codex CLI](https://openai.com/blog/openai-codex)** - OpenAI's Codex model for code generation via command line.

**Licensing:** Proprietary

**Benefits:**
- Powered by OpenAI's code-specialized models
- Natural language to code translation
- Multi-language code generation
- Integrated with GitHub Copilot ecosystem
- Strong understanding of programming patterns and best practices

---

### Droid CLI
**[Droid](https://github.com/droidpl/droid)** - AI-powered assistant for Android development.

**Licensing:** Open Source

**Benefits:**
- Specialized for Android development workflows
- Assists with Gradle configurations and Android-specific tasks
- Reduces boilerplate code generation
- Helps with debugging Android-specific issues
- Community-driven development

---

### Gemini CLI
**[Gemini CLI](https://ai.google.dev/gemini-api)** - Google's Gemini AI accessible through command-line tools.

**Licensing:** Freemium (API-based)

**Benefits:**
- Multimodal capabilities (text, code, images)
- Fast response times
- Competitive free tier for developers
- Integration with Google's AI ecosystem
- Strong performance on coding tasks
- Various community CLI implementations available

---

### Goose CLI
**[Goose](https://github.com/square/goose)** - Open-source AI coding assistant by Square.

**Licensing:** Open Source

**Benefits:**
- Fully open-source and transparent
- Privacy-focused with local execution options
- Extensible architecture for custom plugins
- No vendor lock-in
- Community-driven features and improvements
- Cost-effective for teams

---

### Grok CLI
**[Grok](https://x.ai/)** - xAI's conversational AI accessible through CLI interfaces.

**Licensing:** Proprietary

**Benefits:**
- Real-time information access
- Conversational interface for coding assistance
- Integration with X platform for latest updates
- Emerging AI model with unique capabilities
- Available through community CLI wrappers

---

### OpenCode
**[OpenCode](https://github.com/opencodeiiita/opencode)** - Open-source coding assistant CLI.

**Licensing:** Open Source

**Benefits:**
- Fully open-source alternative
- Community-driven development
- Transparent AI operations
- Customizable for specific workflows
- No API costs or rate limits
- Educational resource for AI/ML enthusiasts

---

### Qodo Command
**[Qodo Command (formerly Codium)](https://www.qodo.ai/)** - AI-powered code integrity platform with CLI support.

**Licensing:** Freemium

**Benefits:**
- Focus on code testing and quality
- Automated test generation
- Code review and suggestions
- Integration with popular IDEs and CI/CD
- Free tier for individual developers
- Premium features for teams
- Emphasis on code reliability and maintainability

---

### Warp CLI
**[Warp](https://www.warp.dev/)** - Modern terminal with built-in AI command search and suggestions.

**Licensing:** Freemium

**Benefits:**
- Native terminal application with modern UX
- AI-powered command suggestions and explanations
- Block-based command output for better readability
- Collaborative features for team workflows
- Cross-platform support (macOS, Linux)
- Free for individual developers
- Built-in AI command search that understands natural language
- Workflow automation and command history search
